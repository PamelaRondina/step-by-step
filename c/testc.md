# Declaração: int

4 prefixos podem ser utilizados:

* **short** -> inteiro pequeno (2 bytes);
* **long** -> inteiro grande (4 bytes);
* **signed** -> inteiro com sinal (negativos e positivos);
*  **unsigned** -> inteiro sem sinal (apenas positivos).
